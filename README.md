# Portal PMESP

Site institucional modelo criado em HTML, CSS e JavaScript.

## Arquivos necessários

- `index.html`
- `style.css`
- `script.js`

## Como publicar no GitHub Pages

1. Entre no repositório `portal-pmesp`.
2. Clique em **Add file > Upload files**.
3. Envie os três arquivos.
4. Clique em **Commit changes**.
5. Vá em **Settings > Pages**.
6. Em **Source**, selecione **Deploy from a branch**.
7. Escolha `main` e `/root`.
8. Clique em **Save**.

Depois o GitHub vai gerar o link público do site.
